 CA_Pref_SetEOL (EOLchars)
 CA_Pref_SetEOL (text)
 
 DESCRIPTION
   This method sets the end-of-line character that is used
   when exporting methods. 

   Permited values are:
    - an empty string; resets to platform default
    - Char(Carriage Return)
    - Char(Line feed)
    - Char(Carriage Return)+Char(Line feed)
